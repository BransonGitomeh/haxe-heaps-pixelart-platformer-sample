# What is this

This is a haxe platformer example using the heaps.io game engine

# Development
prerequisites to install
- node js lts
- haxe 4
- yarn
- hashlink (if not using web)

# Releases
for web releases
```
$ haxe build.js.hxml
# serve the build dir using a file sever
```


for native builds
```
$ haxe build.hl.hxml
# run the .hl file thats created on the build dir
```
